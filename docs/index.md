## ⛓ 区块链（Blockchain）

关注区块链作为一种 **去中心化计算系统** 的底层机制与扩展路径。

主要主题包括：
- Bitcoin：区块结构、UTXO、共识机制
- Lightning Network：链下协议、支付通道、安全性
- EVM：执行模型、Gas 机制、智能合约
- 共识与扩展：PoW / PoS / Rollup / 模块化区块链
